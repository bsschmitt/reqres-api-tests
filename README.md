API Testing Project – ReqRes
Overview
This project contains API tests created using Postman for the ReqRes public API.
Test Coverage
GET Users
POST Login (success & failure)
POST Create User
PUT Update User
DELETE User

Tools Used
Postman
JavaScript (Postman Scripts)

How to Run
Import the collection into Postman
Set environment variables:
base_url = https://reqres.in
api_key = your_api_key

Run requests manually or via Collection Runner

Validations
Status codes
Response structure
Data validation
Negative scenarios
